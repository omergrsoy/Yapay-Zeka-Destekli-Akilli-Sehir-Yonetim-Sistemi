# Yapay-Zeka-Destekli-Akilli-Sehir-Yonetim-Sistemi
Yol haritam, SUMO ile hazır bir veri seti verileri kullanılarak trafik kavşak sistemi oluşturulup kullanılacak (Kırıkkale Yenişehir). Trafik ışıkları, araç yoğunluğu gibi bileşenlerle oluşturulacak. Bağlantı; Python-TraCI ile olucak. Bu veriler Python’da işlenip basit bir Yapay Zeka sistemiyle var olan kavşak ve yollar optimize edilecek.
Gerçek zamanlı trafik optimizasyonu için yapay zeka tabanlı karar sistemi geliştirdim. SUMO simülasyonu ile trafik verisi üretimi ve analizi gerçekleştirdim. Python ve SUMO kullanarak trafik yoğunluğu tespiti geliştirdim. Trafik ışıklarını dinamik olarak optimize eden karar algoritmasını Python ile tasarladım. Simülasyon ortamında gerçek zamanlı sistem entegrasyonunu ile sağladım. Kullanılan Teknolojiler: Python, SUMO
